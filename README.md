## SMARTRACK

## Deskripsi

SMARTRACK adalah aplikasi mobile berbasis Flutter yang digunakan untuk memantau posisi bus jemputan karyawan industri secara real-time.

Tujuan aplikasi adalah membantu karyawan mengetahui posisi bus sehingga waktu tunggu menjadi lebih efisien.

---

## Target Pengguna

1. Karyawan
2. Driver
3. Admin Perusahaan

---

## Role

### Karyawan

- Login
- Dashboard
- Lacak Bus
- ETA
- Rute Bus
- Profil

### Driver

- Login
- Mulai Perjalanan
- Update Lokasi
- Selesai Perjalanan

### Admin

- Dashboard
- Kelola Bus
- Kelola Driver
- Kelola Karyawan
- Monitoring Armada
